# PickCart

PickCart is a modern full-stack e-commerce platform built with Next.js, Express.js, MongoDB, and Better Auth. It provides a complete shopping experience for customers and a product management dashboard for sellers.

## Live Demo

**Client:** https://pickcart-client.vercel.app

**Server:** https://pickcart-server.onrender.com

## Features

### Customer

- User authentication
- Browse products
- Product details
- Cart management
- Checkout UI
- Customer dashboard

### Seller

- Seller dashboard
- Add, update, delete products
- Manage seller products
- Product statistics

## Tech Stack

**Frontend**

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- HeroUI
- Framer Motion

**Backend**

- Node.js
- Express.js
- TypeScript
- MongoDB
- Better Auth
